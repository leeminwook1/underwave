# Underwave (언더웨이브) - 회사 홈페이지

20대 트렌드 매거진 GRAB을 운영하는 마케팅 에이전시 Underwave의 공식 홈페이지입니다.

## 🎨 디자인 컨셉

- **색상**: 흰색 배경에 핑크 포인트 컬러
- **스타일**: 모던하고 깔끔한 디자인
- **반응형**: 모바일, 태블릿, 데스크톱 모두 지원

## 🏢 회사 소개

Underwave는 20대 소비자의 실제 일상과 문화 데이터를 기반으로 트렌드를 분석하고 전략적 인사이트를 제공하는 마케팅 회사입니다.

### 주요 사업

1. **컨설팅 기반 수익** - 마케팅·광고·홍보 전단 컨설팅
2. **마케팅 대행 수익** - SNS 콘텐츠 기획 및 운영
3. **트렌드 리서치 리포트** - 20대 트렌드 분석 리포트
4. **브랜디드 콘텐츠** - 인스타그램 매거진 기반 협업

## 📦 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
```
